# CAPP Agent Recipes

Welcome to the CAPP Agent Recipes repository. This collection contains production-ready, extensible AI agent implementations that demonstrate real-world utility of the CAPP Network.

## Quickstart
Clone this repository and explore the recipes:

- [Recipe 01: Cross-Border Supplier Payment](./recipes/01-supplier-payment/README.md)
- [Recipe 02: Treasury Rebalancing](./recipes/02-treasury-rebalancing/README.md)
- [Recipe 03: Payroll Distribution](./recipes/03-payroll-distribution/README.md)
- [Recipe 04: Payment-Triggered Workflow](./recipes/04-payment-triggered-workflow/README.md)

Each recipe can be run in **Sandbox Mode** without spending real funds.

## Prerequisites
- Python 3.10+
- `capp-sdk` installed via `pip install capp-sdk`
- A CAPP API Key (Sandbox or Production)
